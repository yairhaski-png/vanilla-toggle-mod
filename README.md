# Vanilla Toggle Mod - Minecraft 26.1.2

Fabric mod: לחיצת F12 = הסתרת Resource Packs ו-Shaders (Client-side בלבד).

## בנייה אוטומטית (GitHub Actions)
1. צור repo חדש ב-GitHub (Public או Private)
2. גרור את כל תוכן התיקייה הזו (כולל .github!) לתוך "Add file > Upload files"
3. Commit
4. עבור ל-Actions tab - הבנייה תתחיל אוטומטית
5. אחרי ~2-3 דקות, לחץ על ה-run שהושלם > Artifacts > הורד "vanilla-toggle-mod-jar"
